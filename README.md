## About The Project
An educational project dedicated to a testing of a web-app

## Built With
Java: 11.0.19  
Apache Maven: 3.9.1  
Junit: 4.13.2  
Selenium: 4.9.1  
Allure 2.23.1  
ChromeDriver: 116.0.5845.96  
YandexDriver: 23.7.0  

## HowTo
Run project:
```shell
mvn clean test
```
Prepare Allure test report:
```shell
allure serve target/allure-reports/
```
